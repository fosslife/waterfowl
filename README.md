# Waterfowl

Waterfowl is a database management tool for PostgreSQL. 

| App is work in progress, not ready for production use

## Installation

```bash
yarn install
yarn tauri dev
```

## Features

- Connection Management
- Schema Browser
- Table Browser
- Query Editor
- Data Export
- Data Import
- Data Editing
